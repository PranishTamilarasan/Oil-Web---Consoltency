📦 WEB BASED E-COMMERCE AND INVENTORY MANAGEMENT SYSTEM
📌 Project Description

This project is a web-based application developed to manage coconut-based product sales, inventory, oil mill booking services, and equipment rental in an efficient and digital way. The system provides a user-friendly platform for customers and administrators to interact and manage business operations smoothly.

🎯 Objectives
To digitize product sales and inventory management
To provide online booking for oil mill services
To enable equipment rental management
To improve accuracy and reduce manual work
To enhance user experience and system efficiency

🛠️ Technologies Used
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Other Tools: REST APIs, JSON
👥 User Roles

👤 User
Register and login
Browse products
Place orders
Book oil mill services
Rent equipment
View order, booking, and rental status

👨‍💼 Admin
Manage products (Add/Update/Delete)
Monitor inventory
View and update orders
Manage bookings and rentals
Access dashboard
